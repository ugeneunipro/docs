# Pasting Element

You can paste [_workflow elements_](/workflow-designer/introduction/workflow-elements-and-connections/) that have been [_cut_](/workflow-designer/manipulating-element/cutting-element/) or [_copied_](/workflow-designer/manipulating-element/copying-element/).

To do this, choose the _Paste_ item in the _Actions_ main menu or in the [_Scene_](/workflow-designer//introduction/workflow-designer-window-components/) context menu.

Alternatively, use the Ctrl+V hotkey to paste the elements.
