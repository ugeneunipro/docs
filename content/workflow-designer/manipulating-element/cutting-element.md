# Cutting Element

To cut one or several [_workflow elements_](/workflow-designer/introduction/workflow-elements-and-connections/), select them on the [_Scene_](/workflow-designer/introduction/workflow-designer-window-components/). Choose the _Cut_ option in the _Actions_ main menu or in the selected element's context menu.

The Ctrl+X hotkey is also available for this action.

Now you can [_paste_](/workflow-designer//manipulating-element/pasting-element/) these elements.
