# Bowtie

_Bowtie_ is a popular short read aligner. Click [this link](http://bowtie-bio.sourceforge.net/index.shtml) to open the _Bowtie_ homepage. _Bowtie_ is integrated as an [_external tool_](/extensions/external-tools-plugin/) into UGENE.

Open the _Tools ‣ NGS data analysis_ item submenu of the main menu.

![](/images/65930850/82608135.png)

Select the _Map reads to reference_ option to align short reads to a DNA sequence using _Bowtie_. Alternatively, select the _Build index to reads mapping_ option to create an index for a DNA sequence, which can be used to improve the efficiency of aligning the short reads to the sequence.
