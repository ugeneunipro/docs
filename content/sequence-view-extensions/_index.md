# Sequence View Extensions

The functionality of the _Sequence View_ can be significantly enhanced with _Sequence View Extensions_. Below is a demonstration of its functionality.

The [_Circular Viewer_](/sequence-view-extensions/circular-viewer/) displays the circular view of a sequence:

![](/images/65929506/65929507.png)

The [_3D Structure Viewer_](/sequence-view-extensions/3d-structure-viewer/) adds 3D visualization for PDB and MMDB files:

![](/images/65929506/65929508.png)

The [_Chromatogram Viewer_](/sequence-view-extensions/chromatogram-viewer/) adds support for the visualization and editing of chromatograms:

![](/images/65929506/65929509.png)

The [_DNA Graphs Package_](/sequence-view-extensions/graphs-package/) displays various graphs for sequences:

![](/images/65929506/65929510.png)

The [_Dotplot_](/sequence-view-extensions/dotplot/) provides a tool to create dot plots for DNA or RNA sequences.

![](/images/65929506/65929511.png)

Several other instruments add a graphical interface for popular sequence analysis methods:

![](/images/65929506/65929512.png)
