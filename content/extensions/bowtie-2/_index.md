# Bowtie 2

_Bowtie 2_ is a popular, ultrafast, and memory-efficient tool for aligning sequencing reads to long reference sequences. Click [this link](http://bowtie-bio.sourceforge.net/bowtie2/index.shtml) to open the _Bowtie 2_ homepage. _Bowtie 2_ is embedded as an [_external tool_](/extensions/external-tools-plugin/) into UGENE.

Open the _Tools ‣ NGS data analysis_ submenu of the main menu.

![](/images/65930857/82608151.png)

Select the _Map reads to reference_ item to align short reads to a DNA sequence using Bowtie 2.

Alternatively, select the _Build index to reads mapping_ option for a DNA sequence, which can be used to optimize the alignment of the short reads to the sequence.
