# Trimming the Right End

To trim the right end in the _Sanger Reads Editor_, select the symbol immediately before the portion you wish to trim, then use _Ctrl+Shift+Del_ or select _Edit_â€“>_Trim right end_ from the context menu to activate the edit mode.

After trimming, the edit mode is automatically switched off.

You can always cancel the changes by pressing _Ctrl+Z_.
