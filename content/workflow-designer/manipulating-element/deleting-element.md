# Deleting Element

Select one or several [_workflow elements_](/workflow-designer/introduction/workflow-elements-and-connections/) on the [_Scene_](/workflow-designer/introduction/workflow-designer-window-components/) that you want to delete. Then choose the _Delete_ item in the _Actions_ main menu or in the selected element's context menu.

The hotkey for this action is Del.
