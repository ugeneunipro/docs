# Selecting All Elements on Scene

To select all [_workflow elements_](/workflow-designer/introduction/workflow-elements-and-connections/) present on the [_Scene_](/workflow-designer//introduction/workflow-designer-window-components/), choose _Select all elements_ in the _Actions_ main menu or in the Scene context menu.

Alternatively, use the Ctrl+A hotkey.
